# Task 4: Feature Encoding & Scaling
## Internship: AI & ML Internship
### Dataset
Adult Income Dataset (UCI Machine Learning Repository)

### Objective
To perform feature encoding and scaling to prepare data for machine learning models.

### Steps Performed
- Loaded Adult Income dataset in Google Colab
- Handled missing values
- Identified categorical and numerical features
- Applied Label Encoding where order exists
- Applied One-Hot Encoding for nominal features
- Scaled numerical features using StandardScaler
- Compared data before and after scaling
- Saved the processed dataset

### Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

### Output Files
- Task_4_Feature_Encoding_and_Scaling.ipynb
- adult_income_processed.csv
